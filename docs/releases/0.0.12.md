# SumUp Terminal for WooCommerce 0.0.12

## Checkout reader loading fix

- Retries reader discovery when the cashier opens SumUp after another payment method was initially selected.
- Reloads only an empty SumUp payment panel, leaving an already-rendered reader selection unchanged.
- Retries after WooCommerce refreshes the checkout payment methods.

## Compatibility notes

- The minimum supported PHP version remains PHP 7.4.
